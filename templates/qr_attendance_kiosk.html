  1. USB RFID Reader — Plug any HID-compatible RFID reader into the kiosk device. It emulates keyboard input — the code goes directly into the Member Code field above.
  2. RFID Card Programming — Program each card with the staff member's member code (e.g. SHRQ001). Use the card writer software from Ashim Patil's company.

  4. Mobile QR Alternative — Staff can show the QR from "My QR Code" page on their phone. The kiosk operator scans it with a camera or handheld scanner.

Supported hardware: Any USB HID barcode/RFID reader. Recommended: Ashim Patil's EM-4100 readers with CR suffix (auto-Enter after read). No drivers needed — plug and play on any OS.
