{{ card_heading("HOW TO EXPORT FROM TALLY", "Step-by-step") }}
  1. Open Tally Prime on your Windows machine
  2. Go to Gateway of Tally > Display More Reports > Day Book
  3. Set the date range for the period you want to reconcile
  4. Press Ctrl+E or click Export
  5. Choose CSV (Comma Delimited) format
  6. Save the file and upload it here

Expected columns: Date, Voucher Type, Voucher No, Narration, Debit, Credit, Ledger Name. The system auto-detects common Tally export formats.
